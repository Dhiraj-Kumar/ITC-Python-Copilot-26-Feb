# Personal Profile Website

This project is a personal profile website for a freelance software developer. It showcases the developer's skills, projects, and provides contact information for potential clients. The website features a modern and aesthetic design, ensuring a pleasant user experience.

## Project Structure

```
personal-profile-website
├── src
│   ├── index.html          # Main HTML file serving as the entry point
│   ├── styles
│   │   └── main.css       # CSS file for styling the website
│   ├── components
│   │   ├── navbar.html     # Navigation bar component
│   │   ├── about.html      # About section component
│   │   ├── projects.html   # Projects section component
│   │   └── contact.html    # Contact section component
│   └── assets
│       └── images
│           └── README.md   # Placeholder for image assets
└── README.md               # Documentation for the project
```

## Features

- **Navigation Bar**: A responsive navigation bar at the top for easy access to different sections of the profile.
- **Smooth Scrolling**: Bookmarks for smooth scrolling between sections.
- **Modern Design**: Aesthetic layout with a focus on usability and readability.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the website.
3. Customize the content in the `about.html`, `projects.html`, and `contact.html` files as needed.

## Technologies Used

- HTML5
- CSS3
- Responsive Design Principles
